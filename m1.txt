1. This project implements **real-time hand detection using Python**.
2. It uses a **webcam to capture live video frames**.
3. The project uses the **OpenCV library for video processing**.
4. It uses **MediaPipe Hands solution** for detecting hand landmarks.
5. The system can detect **up to two hands simultaneously**.
6. Each detected hand contains **21 landmark points**.
7. The landmarks represent important points like **wrist and finger joints**.
8. The coordinates of each landmark **(x, y)** are printed in the terminal.
9. The landmarks and their **connections are drawn on the video frame**.
10. Frames are converted from **BGR to RGB format** before processing.
11. Detection works **continuously in a loop for real-time tracking**.
12. The output window displays the **live hand detection result**.
13. The program stops when the user **presses the 'q' key**.
14. This project can be extended to build **gesture control applications**.
15. It can be used for **virtual mouse, volume control, or sign language recognition systems**.
